# 勤怠管理アプリ（attendance-app）

## 概要
本プロジェクトは、Spring Boot を使用した勤怠管理アプリの開発プロジェクトです。  
要件定義から設計、実装、テストまでの一連の開発工程を実践することを目的としています。

本アプリでは以下のような基本的な勤怠管理機能を実装予定です。

- 出勤打刻
- 退勤打刻
- 休憩開始／終了
- 日次勤怠一覧表示
- 月次勤怠一覧表示
- 勤務時間自動計算

---
